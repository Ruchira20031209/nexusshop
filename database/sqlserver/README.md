# SQL Server Setup

Run the scripts in this order:

1. `01-core-schema.sql`
2. `02-cart-faq.sql`
3. `03-orders.sql`

Notes:

- `legacy-schema.txt` is kept only as a reference copy from the original project.
- The application expects a database named `NexusShop2`.
- For Azure SQL Database, run the scripts in the Azure portal query editor, Azure Data Studio, or SSMS.
